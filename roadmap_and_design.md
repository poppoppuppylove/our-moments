# LifeLog - 生活记录网站 Roadmap & Design

## 1. 项目愿景 (Vision)

打造一个温馨、自由、充满手绘质感的个人生活记录空间。不仅仅是博客，更是线上的“拼贴手账”。用户可以像在纸质手账上一样，自由排版文字、图片、贴纸，记录生活的点滴瞬间。

**核心关键词：** 手绘风、拼贴、自由排版、温暖、低饱和度

## 2. 视觉风格指南 (Visual Identity)

### 2.1 核心风格
*   **模拟手绘 (Hand-drawn):** 所有边框、分割线、图标都应带有手绘的轻微不规则感，避免绝对的直线和正圆。
*   **拼贴艺术 (Collage):** 图片和便签的展示方式模拟实物粘贴，带有轻微的角度倾斜、阴影和层叠感。
*   **纸质质感 (Paper Texture):** 背景可使用轻微的纸张纹理（如水彩纸、牛皮纸），增加温暖感。

### 2.2 配色方案 (Color Palette)
采用低饱和度、柔和的自然色系：
*   **背景色:**
    *   `#F9F5F0` (米白 - 主背景，模拟纸张)
    *   `#F0EAD6` (蛋壳色 - 次级背景)
*   **线条与文字:**
    *   `#4A4A4A` (炭灰 - 深色文字/主线条，比纯黑更柔和)
    *   `#8C8C8C` (中灰 - 辅助文字)
*   **点缀色 (低饱和):**
    *   `#E0BBE4` (淡 dusty pink)
    *   `#957DAD` (柔和紫)
    *   `#D291BC` (浅藕合)
    *   `#FEC8D8` (淡桃红)
    *   `#FFDFD3` (淡粉)
    *   `#B5EAD7` (薄荷绿 - 少量点缀)

### 2.3 字体 (Typography)
*   **标题:** 选用手写体风格的中文字体（如：演示悠然小楷、字制区喜脉体、或者类似 Google Fonts 的 'Ma Shan Zheng'）。
*   **正文:** 清晰易读的衬线体或圆体（如：Noto Serif SC, 霞鹜文楷），保持阅读舒适度的同时不失人文气息。

### 2.4 界面元素 (UI Elements)
*   **图片容器:**
    *   拍立得风格（宽白边 + 下方留空）。
    *   胶带粘贴效果（图片角落有半透明胶带素材覆盖）。
    *   不规则裁切边缘。
*   **阴影:** 使用弥散的、带有色彩倾向的暖色阴影，而非生硬的黑色投影。

## 3. 功能 Roadmap

### Phase 1: 核心基础 (MVP)
*   **目标:** 搭建基础框架，实现核心的“手账式”阅读体验。
*   **功能:**
    *   **首页:** 瀑布流或时间轴展示生活记录（Log），每个卡片都是独特的拼贴样式。
    *   **详情页:** 支持富文本，图片可自由插入（基础版），不仅限于标准网格，图片允许有预设的旋转角度。
    *   **后台管理:** 基础的增删改查 (CRUD) 文章和图片。
    *   **后端:** Spring Boot 搭建 RESTful API，数据库设计 (MySQL)。
    *   **前端:** Vue 3 + Vite 搭建，实现基础的手绘风组件库。

### Phase 2: 自由排版编辑器 (The "Collage" Editor)
*   **目标:** 实现“操作更灵活、贴进去的感觉”。
*   **功能:**
    *   **拖拽排版:** 在编辑器中，用户可以拖拽图片位置，而不仅仅是行内插入。
    *   **自由变换:** 图片/贴纸支持缩放、旋转 (Rotate)。
    *   **装饰素材库:** 提供预设的手绘胶带、图钉、涂鸦线条贴纸，用户可拖拽到文章中。
    *   **层级控制:** 控制图片、文字、贴纸的前后遮挡关系 (Z-index)。

### Phase 3: 互动与丰富
*   **目标:** 增加网站的活力和内容丰富度。
*   **功能:**
    *   **标签/分类系统:** 用“手绘标签牌”的形式展示。
    *   **日历视图:** 手绘风格日历，点击日期查看当天的记录。
    *   **评论系统:** 模拟“留言条”或“便利贴”的样式。
    *   **移动端适配:** 优化在手机上的触摸和阅读体验。

## 4. 技术栈 (Tech Stack)

### 前端 (Frontend)
*   **框架:** Vue 3 (Composition API)
*   **构建工具:** Vite
*   **语言:** TypeScript
*   **样式:** SCSS / TailwindCSS (配合自定义配置实现低饱和配色)
*   **拖拽/交互库:** `moveable` 或 `interactjs` (实现图片的拖拽、旋转、缩放), `vue-draggable-next`
*   **状态管理:** Pinia

### 后端 (Backend)
*   **框架:** Spring Boot
*   **语言:** Java (JDK 17+)
*   **数据库:** MySQL
*   **ORM:** MyBatis
*   **对象存储:** 阿里云 OSS (存储图片、视频资源)

## 6. 关键 UI 实现思路

### 6.1 “轻粗糙”线条实现
不要使用默认的 `border: 1px solid black`。
*   **CSS `border-image`:** 使用手绘线条的 SVG 作为边框图片。
*   **SVG Filters:** 使用 SVG 滤镜给 `div` 增加波浪形或噪点边缘。
*   **CSS `box-shadow` 多重投影:** 模拟不均匀的笔触。
*   **Rough.js:** (可选) 一个 JS 库，专门用于绘制手绘风格的基础图形。

### 6.2 图片“贴进去”的效果
为图片容器添加随机的旋转角度：
```css
.photo-container {
  box-shadow: 2px 4px 10px rgba(0,0,0,0.1); /* 柔和阴影 */
  border: 4px solid #fff; /* 宽白边模拟照片 */
  transform: rotate(-2deg); /* 随机微小旋转 */
  transition: transform 0.3s ease;
}
.photo-container:hover {
  transform: rotate(0deg) scale(1.02); /* 悬停交互 */
}
```

### 6.3 胶带效果
使用 `::before` 或 `::after` 伪元素，加载半透明的胶带 PNG 背景图，定位在图片的右上角或正上方。
